# ToyML-Zero
A toy-implementation of [AutoML-Zero](https://github.com/google-research/google-research/tree/master/automl_zero) written in Python (without efficient implementation).

## Current Status
WIP

## How to use
```console
$ pipenv install
$ pipenv run python res.py
```
